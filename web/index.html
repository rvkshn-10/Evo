Neural Compute Stick (local Mac)

Plug an Intel NCS into USB on the Mac running python3 main.py — not available on the cloud server.

Checking runtime…

  1. Use the local dashboard — open http://localhost:5173 (not Vercel) while the API runs on your Mac.
  2. Install OpenVINO (driver for NCS — optional on CPU): 
    pip install openvino
    OpenVINO is Intel’s runtime that routes inference to the Neural Compute Stick or CPU. You do not need it on the Vercel site.
  3. Plug in your stick — NCS2 (recommended) or NCS1 via USB. Use a powered hub if needed.
  4. Confirm model files exist: 
    models/evo1.2/openvino/evo1.2.xml
models/evo1.2/openvino/evo1.2.bin
  5. Start the API and pick your device from the dropdown: 
    python3 main.py
    Select NCS2 or NCS1 in Inference device — no restart needed when you plug the stick in later; click Check again.

Without a stick, choose CPU or Auto — Evo runs on the server CPU. The Vercel URL always hits Oracle cloud (no USB). For NCS, run locally at localhost:5173. See docs/NCS_SETUP.md in the repo.
